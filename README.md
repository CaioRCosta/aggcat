# aggcat

> A simple CLI to aggregate code analysis tools and metrics in one place.

---

## What is it?

**aggcat** is a lightweight command-line tool that helps you run multiple code analysis tools together.
It makes it easier to spot code smells, track quality metrics, and keep your codebase clean.

---

## Why use it?

* 🔍 Detect issues faster
* 📊 Centralize metrics
* 🧩 Avoid tool fragmentation
* 🚀 Improve code quality with less effort

---

## Usage

```bash id="n4g2k1"
aggcat
```
